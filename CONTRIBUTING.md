# Contributing

This repository is a personal, curated reference list. Changes should keep it compact, trustworthy, and easy to scan.

## Adding or replacing a link

1. Prefer official documentation or the project's primary repository.
2. Use HTTPS.
3. Confirm that the page is current and actively maintained.
4. Add the link to the most specific existing section.
5. Replace obsolete or duplicate links rather than preserving them indefinitely.
6. Keep link text descriptive; avoid labels such as `click here`.

## Branch and pull request policy

- `main` is the only long-lived branch.
- Create a short-lived branch for each change.
- Submit changes through a pull request.
- Delete the source branch after merge.
- Do not merge a pull request until its link-check workflow succeeds or any exception is documented.
