## Scratch

Scratch is primarily a browser-based educational language and environment.

| Platform | Official installation |
| --- | --- |
| Windows | [Scratch app download](https://scratch.mit.edu/download) |
| macOS | [Scratch app download](https://scratch.mit.edu/download) |
| Linux | [Scratch online editor](https://scratch.mit.edu/projects/editor/) — no official native Linux desktop installer |

The online editor is also the most portable route for Windows and macOS when offline use is not required.
